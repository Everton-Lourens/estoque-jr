# Rodar frontend localmente com Node

## Passos
1. Acesse a pasta `frontend/`.
2. Instale as dependências com `npm install`.
3. Inicie o servidor local com `npm start`.
4. Abra `http://localhost:3000/` no navegador.

## Observação
- O servidor apenas entrega a mesma pasta estática usada no GitHub Pages.
- O `index.html` continua sendo a entrada principal e o diretório `sucess/` continua acessível como rota estática.
