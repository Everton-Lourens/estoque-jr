# Publicar o site no GitHub Pages

## Passos
1. Suba os arquivos estáticos para o repositório.
2. Confirme o `appScriptUrl` em `config.js`.
3. Habilite o GitHub Pages na branch desejada.
4. Abra a URL publicada e valide se o bootstrap carrega funcionários e itens.
5. Se a leitura direta falhar, confira se o fallback JSONP continua retornando o bootstrap.
6. Faça um envio de teste e confira a gravação na planilha.

## Resultado esperado
O frontend consegue carregar os dados iniciais do formulário e o backend consegue gravar pedidos.
